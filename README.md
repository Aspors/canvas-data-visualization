# 📈 Canvas Log Chart Engine

Высокопроизводительный, независимый от фреймворка (framework-agnostic) Canvas-движок для рендеринга вертикальных графиков (каротаж, телеметрия, скважинные данные).

Спроектирован для плавной работы с огромными массивами данных (сотни тысяч и миллионы точек) за счет многослойного рендеринга и эффективных алгоритмов пространственного поиска.

## ✨ Ключевые особенности

- **Framework-Agnostic:** Ядро написано на чистом TypeScript. Идеально интегрируется в любые фреймворки (Vue, React, Angular) или Vanilla JS.
- **Вертикальная ориентация:** Нативная поддержка графиков, где независимая переменная (глубина/высота) располагается по оси Y.
- **Многослойный рендеринг (Multi-canvas):** Разделение на статичные (сетка), тяжелые (данные) и динамичные (UI, тултипы) слои для обеспечения 60 FPS при взаимодействии.
- **Умный Render Pipeline:** Перерисовка слоев происходит только при их явной инвалидации (`needsUpdate`), исключая лишнюю нагрузку на CPU.
- **$O(\log n)$ Интерактивность:** Использование бинарного поиска по отсортированной глубине для мгновенного нахождения точек под курсором.
- **Паттерн Адаптер:** Готовность к интеграции с легаси-форматами бэкенда (например, парсинг серий Highcharts).
